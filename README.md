
```
https://git-scm.com
```

### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://github.com/Violeta1977/hangman-react
    ```
2. Open folder named "hangman-react"

3. Install NPM packages
    ```sh
    npm i
    ```
    or
    ```sh
    npm install
    ```
4. Run the server
    ```sh
    npm run dev
    ```
